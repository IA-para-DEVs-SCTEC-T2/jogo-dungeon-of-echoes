## Arquivos necessários

Extraia o tileset e coloque os arquivos abaixo nos respectivos diretórios:

```
public/assets/dawnlike/
  Objects/
    Ground0.png   ← tiles de chão (pedra, terra, etc.)
    Wall0.png     ← tiles de parede
  Characters/
    Player0.png   ← sprites do personagem jogável
    Undead0.png   ← sprites de inimigos (esqueletos)
    Reptiles0.png ← sprites de répteis (contém o Platino — easter egg obrigatório)
```
